## 简介

client-init-template 是一个非常清爽的前端程序，基于 Vue3、Vite、TypeScript、Element Plus 等主流技术，但是除了基础的权限都只是空目录，可自主卸载或者删除

## 内置功能

**用户管理**：登录、登出演示

**权限管理**：页面级权限（动态路由）、按钮级权限（指令权限、权限函数）、路由守卫

**首页**：根据不同用户显示不同的 Dashboard 页面

**错误页**: 403、404

**兼容移动端**: 布局兼容移动端页面分辨率

**其他**：SVG 雪碧图、动态侧边栏、动态面包屑、标签页快捷导航、内容区放大与全屏、组合式函数

## 技术栈

**Vue3**：采用 Vue3 + script setup 最新的 Vue3 组合式 API

**Element Plus**：Element UI 的 Vue3 版本

**Pinia**: 传说中的 Vuex5

**Vite**：真的很快

**Vue Router**：路由路由

**TypeScript**：JavaScript 语言的超集

**pnpm**：更快速的，节省磁盘空间的包管理工具

**CSS 变量**：主要控制项目的布局和颜色

**Axios**：发送网络请求（已封装好）

**UnoCSS**：具有高性能且极具灵活性的即时原子化 CSS 引擎
